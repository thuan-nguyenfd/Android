# Hướng dẫn lấy API key thế vào file Manifest

### Bước 1:
Truy cập <a>https://console.cloud.google.com/</a>
<br>
### Bước 2:
Tại navigation menu chọn `APIs & Services` và chọn `Library` sau đó chọn `Maps SDK for Android` và `Enable`
<br>
### Bước 3:
Tại menu chọn `Credentials` và chọn `Create Project` <i>nếu chưa có (Yêu cầu tài khoản bảo mật 2 lớp)<i> để có thể xuất hiện nút `Create credentials` rồi chọn `API key`.<br>
